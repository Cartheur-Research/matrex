# Matrex

Matrix manipulation library for Elixir implemented in C native code.
Extracted from https://github.com/sdwolfz/exlearn

## Installation

The package can be installed
by adding `matrex` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:matrex, "~> 0.1.0"}
  ]
end
```
